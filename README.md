# Portfólio • Aryelle Araújo

Este repositório contém o desenvolvimento do portfólio de **Aryelle Araújo**, criado para apresentar suas informações profissionais, projetos e formas de contato em uma interface simples, moderna e responsiva.

## Sobre o projeto

- Desenvolvimento de um site de portfólio personalizado;
- Layout responsivo para computadores e dispositivos móveis;
- Estrutura organizada para apresentação profissional;
- Interface limpa e de fácil navegação.

## Desenvolvimento

O projeto foi desenvolvido com o auxílio de Inteligência Artificial como ferramenta de apoio durante o processo de criação, sendo utilizado para acelerar o desenvolvimento e auxiliar na implementação de funcionalidades e ajustes.

## Hospedagem

O site foi publicado utilizando uma plataforma de hospedagem gratuita (**Vercel** ou **Netlify**) para facilitar seu acesso e compartilhamento.
